## Accountant app

Is made to be easy customized and deployed. Using [theme-ui](https://theme-ui.com/home)
to achieve highly customizible UI.

### Installation

```
pnpm
```

### Development

Create `frontend/.env` file with the following content:

```dotenv
PUBLIC_URL=/accountant
```

Then run:

```
pnpm start
```

```
pnpm build
```

### UI Customization

Places to look at:

- `theme/*`
- `components/*`
